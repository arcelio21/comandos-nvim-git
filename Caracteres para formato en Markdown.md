# Para Listas no ordenadas  
Se puede utilizar el los siguiente signos:  
- **El signo de  (-) al principio de cada elemento de la lista**
- El signo de (+) ||  
- El signo de  (*) ||  
# Para lista ordenada  
Se utiliza el numero y un punto pegado y luego de un espacio el texto a enumerar
1. Primero
