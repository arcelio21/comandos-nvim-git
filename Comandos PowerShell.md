# Comandos que se pueden utilizar en powershell

## Cambiar nombre de una carpeta

`Renama-Item 'nombre de archivo' 'nuevo nombre de archivo' `

## Crear archivo

`New-Item 'nombre de archivo' `

## Crear carpeta

`mkdir 'Nombre de carpeta' `
